Calculate the cube of a given integer

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Create a function that takes an integer number as an argument and returns the cube of the number. The cube of a number 'n' can be calculated as 'n*n*n'. Your task is to print the cube of the numbers you input.
Sample Input:
2

Sample Output:
8

5
