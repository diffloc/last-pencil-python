Counting words and reversing a string

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Imagine you are given a string that represents a sentence or a phrase. Create a program that scans a string as single line input. Your program should print the word count in that string and the reversed string separated by a newline.
Sample Input:
Hello World

Sample Output:
2
World Hello


Sample Input:
Python is fun

Sample Output:
3
fun is Python



Memory limit: 256 MB
Time limit: 5 seconds
